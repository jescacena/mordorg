# Cercemap App
Mapping application for Cercedilla Points of Interests

## Install
```
npm install
```

## Local development
* Local server
```
npm start
```

* Webpack build
```
webpack
```

* Webpack build with watch
```
webpack -w
```

* Webpack bumping version (--major --minor --patch). Version is bumped in package.json
```
webpack --major
```

## Unit tests
* Run tests
```
npm test
```
